# Задание 6: Админ-панель с HTTP-авторизацией

## Описание
Админ-панель для просмотра, редактирования и удаления данных пользователей, а также статистики по языкам программирования.

## Технологии
- PHP 7+
- MySQL/MariaDB
- HTML5/CSS3
- HTTP Basic Authentication

## Функционал
1. HTTP-авторизация (Basic Auth)
2. Просмотр всех заявок пользователей
3. Редактирование заявок
4. Удаление заявок
5. Статистика по языкам программирования (GROUP BY)

## Структура БД
- `admins` - администраторы (логин, хеш пароля)
- `applications` - заявки пользователей
- `programming_languages` - справочник языков
- `application_languages` - связь заявок и языков

## Установка
1. Импортировать db.sql в MySQL
2. Создать админа: php -r "echo password_hash('admin123', PASSWORD_DEFAULT);"
3. Вставить хеш в таблицу admins
4. Открыть admin.php в браузере
